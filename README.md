# Claude Code Mastery — Lab Repositories

Sample codebases for the [Claude Code Mastery](https://github.com/Claude-Trainer/Claude-Trainer) training program.

## Labs

| Lab | Module | Description | Difficulty |
|-----|--------|-------------|-----------|
| `lab-01-foundations/` | Module 1 | Express + React task manager — your first Claude Code session | Beginner |
| `lab-02-workflows/` | Module 2 | Bug hunt (5 bugs) + feature development exercise | Beginner–Intermediate |
| `lab-03-configuration/` | Module 3 | Build CLAUDE.md, commands, skills, and hooks from scratch | Intermediate |
| `lab-04-advanced/` | Module 4 | Multi-agent refactoring, MCP, CI/CD, Agent SDK | Advanced |

## Quick Start

```bash
# Clone this repo
git clone https://github.com/Claude-Trainer/Claude-Trainer-Assets.git

# Start Lab 1
cd Claude-Trainer-Assets/lab-01-foundations
npm install
npm run dev

# Open Claude Code
claude
```

Each lab has its own README with detailed instructions, completion checklists, and (for instructors) solution hints.

## Requirements

- Claude Code installed ([installation guide](https://code.claude.com/docs/en/setup))
- Claude Pro subscription or higher ($20/month minimum)
- Node.js 18+ (for the lab projects)
- Git
